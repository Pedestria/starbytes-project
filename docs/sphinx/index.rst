Starbytes Documentation
=======================

.. container:: hero-card

   Starbytes is a strongly typed language and toolchain with a bytecode
   compiler/runtime, native standard library modules, compiler-backed LSP
   support, and a shared linguistics layer for formatting, linting, and code
   actions.

   This Sphinx set is organized around the three surfaces that matter most when
   you are working in the repository: the language itself, the shipped tools and
   command-line entrypoints, and the standard library modules exposed to user
   code.
